# Team-Profile-Generator

## Description

This project prompts the user for details on various members in a software development team, then display that information in a dynamically created HTML file.

## Installation

To install necessary dependencies, run the following command:

`npm i`

## Usage

To run this project, cd into the project directory in the command prompt and run the `node index.js` command. You will then be prompted to answer various questions describing a software engineer team. Once you are done adding team members, an index.html file will be created in the '/dist' folder that will display your team information.

Here is a video that walks through how to run the program: https://drive.google.com/file/d/1T_LCMq7xRitzBFq_Ueuj1pW2idHG-kDl/view?usp=share_link
